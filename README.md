# urban-biodiversity-dce
Data and Stan code for urban biodiversity DCE
This repository contains Python and Stan code, data, and documentation for the analysis of willingness-to-pay for biodiversity improvements in urban settings.

## Contents
- `/stan/` — Stan model files  
- `/data/` — Processed data  
- `/notebooks/` — Jupyter notebooks  
- `/scripts/` — Python uscripts
